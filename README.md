# old-homes-finder

Create a website that helps to find old flats and houses. We want to enable our audience to buy affortable homes and renovate them, rather than tearing it down and build something new. We wanna save the old and embrace new techlogies (solar, heat pump, eco isolation).

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/old-homes-finder.git
cd old-homes-finder
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
